# Build

Before doing anything else, run:

```bash
npm install
```

## Develop

### Frontend

```bash
npx shadow-cljs watch pikseli
npx shadow-cljs watch metsa
```

### Backend

```bash
lein repl
```

## Compile production

### Frontend

```bash
npx shadow-cljs release pikseli
npx shadow-cljs release metsa
```

### Backend

```bash
lein uberjar
```
